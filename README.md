# C++ Studies & Algorithms

A professional repository dedicated to the study of C++, data structures, and algorithms. This project aims to track my progress from fundamental language concepts to advanced algorithmic implementations.

## 🚀 Objectives

- Deepen understanding of C++ fundamentals (types, pointers, memory management, OOP).
- Implement classic data structures (Trees, Graphs, Hash Maps, Linked Lists).
- Solve and document algorithmic challenges.
- Adopt professional coding standards and best practices.

## 📂 Repository Structure

- `c++/`: Core C++ language concepts, experiments, and fundamental exercises.
- `binary_tree.cc`: Introduction to tree data structures.
- `.vscode/`: Configuration for seamless development in Visual Studio Code.

## 🛠️ Technologies & Tools

- **Language:** C++ (C++17 / C++20)
- **Compiler:** GCC (g++)
- **Editor:** Visual Studio Code with Code Runner extension

## 📈 Learning Journey

1. **Fundamentals:** Variables, types, scope, and basic I/O.
2. **Control Flow:** Loops and conditional statements.
3. **Memory Management:** Pointers, references, and dynamic memory allocation.
4. **Object-Oriented Programming (OOP):** Classes, inheritance, and polymorphism.
5. **Data Structures:** Implementing common structures from scratch.
6. **Algorithms:** Searching, sorting, and dynamic programming.
